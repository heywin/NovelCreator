# Project Template

Use this compact project layout for a new novel.

```text
project-root/
  PROJECT_ENTRY.md
  00-project-overview.md
  01-cast-bible.md
  02-relationship-map.md
  03-timeline.md
  04-plot-lines.md
  05-chapter-roadmap.md
  06-continuity-ledger.md
  07-style-and-voice.md
  08-active-state.md
  09-literary-revision.md
  chapters/
  chapter-cards/
```

## File Content Hints

### PROJECT_ENTRY.md

- first-read handoff instructions
- required read order
- method or skill location
- current progress
- next intended task
- active facts and forbidden drift
- post-draft update checklist

### 00-project-overview.md

- title
- genre
- scale
- promise
- audience
- ending direction
- taboo boundaries

### 01-cast-bible.md

- name
- role
- age
- surface traits
- hidden pressure
- desire
- fear
- contradiction
- arc hypothesis

### 02-relationship-map.md

- pair or group
- relation type
- current status
- pressure source
- debt or leverage
- likely change

### 03-timeline.md

- date or order
- event
- who knows it
- why it matters
- downstream effect

### 04-plot-lines.md

- line name
- function
- current stage
- next turn
- collision point

### 05-chapter-roadmap.md

- chapter
- purpose
- dominant line
- key beat
- exit condition

### chapter-cards/NN-card.md

- purpose
- entry state
- required beats
- character constraints
- relationship pressure
- timeline and setting constraints
- plot line movement
- foreshadowing and payoff
- narrative boundary
- forbidden drift
- exit state
- post-draft update targets

### 06-continuity-ledger.md

- fact
- source chapter
- evidence status
- status
- risk
- follow-up

### 07-style-and-voice.md

- narration stance
- paragraph behavior
- dialogue behavior
- banned habits
- lexical limits
- narrative boundary rules for manuscript language

### 08-active-state.md

- what changed last
- what is true now
- what must stay true
- what to check next
- active line heat

### 09-literary-revision.md

- batch quality verdict
- strong scenes
- weak scenes
- convenience risks
- interior arc notes
- texture and world notes
- rhythm and length notes
- narrative boundary notes
- required revisions before next batch
- deferred revisions
